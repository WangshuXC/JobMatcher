# 2026-03-24 工作日志

## 创建 JobMatcher 项目

### 技术栈
- Next.js 16 (App Router, TypeScript)
- TailwindCSS v4 + Shadcn UI (base-nova 风格)
- Motion (framer-motion) 动画
- Playwright 爬虫 (headless chromium)

### 项目架构
- `src/types/index.ts` - 核心类型定义 (JobPosting, ResumeData, MatchResult 等)
- `src/lib/store.ts` - 内存数据存储 (单例 jobStore)
- `src/lib/crawler/base.ts` - 爬虫抽象基类 (BaseCrawler)
- `src/lib/crawler/registry.ts` - 爬虫注册表
- `src/lib/crawler/sources/` - 各大厂爬虫实现
  - `bytedance.ts` - 字节跳动 (已完整实现)
  - `tencent.ts` - 腾讯 (预设架构)
  - `alibaba.ts` - 阿里巴巴 (预设架构)
- `src/lib/matcher.ts` - 简历解析与智能匹配引擎
- `src/app/api/` - API 路由 (sources, crawl, jobs, match)
- `src/components/` - 前端组件 (CrawlerPanel, JobList, ResumeMatch)

### 扩展方式
新增数据源：在 `sources/` 下新建类继承 `BaseCrawler`，实现 `crawlJobList` 和 `crawlJobDetail`，然后在 `registry.ts` 中注册。

### 注意事项
- Playwright 配置在 `next.config.ts` 中的 `serverExternalPackages` 排除打包
- 字节跳动招聘页面是 SPA，需要等待 JS 渲染
- 开发服务端口: 3001 (3000 被占用)
